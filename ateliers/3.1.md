# Atelier 3.1 : comprendre les principes de git

1. Clonez les dépôts suivants :
- [https://github.com/vuejs/core](https://github.com/vuejs/core)
<!--
- [https://github.com/facebook/react](https://github.com/facebook/react)
- [https://github.com/angular/angular](https://github.com/angular/angular)
-->

2. Explorez l'historique des dépôts clonés à l'aide des commandes suivantes :
- `git log --oneline`
- `git log --graph --all`

Navigation avec git log :
- touche **entrée** pour voir la ligne suivante ou touche **Haut** et **Bas** pour naviguer
- touche **q** pour quitter

3. Explorez le Web (*github.com*) et clonez un projet git de votre choix.
