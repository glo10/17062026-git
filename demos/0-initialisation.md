# Configuration du poste en local

1. Il faut s'identifier par rapport à GIT avec au minimum les 2 infos suivantes : 
- Votre nom
- Votre email
La commande  git config pour le faire
```bash
git config --global user.name glodie
git config --global user.email contact@tshimini.fr
```

---

## La liste des configs

- Pour voir la liste des configs
```bash
git config --list
```
